Installation Guide Top

How do I install XWIS Multiplayer Games?

Simply download one of the multiplayer clients from our Downloads page and double click to install. If you haven't created a nick, log into your XWIS Game Account and click "Create nick". Start the game using "Run as Administrator" then click on "Internet". Login using the nick create within you game account.

Playing Online Top

What can't I connect to anyone?

This typically means your ports are not forwarded correctly. For all XWIS Games, the following UPD ports need to be open: 1234-1237, 5400, 4000, 5009. Simliarly the follwing TCP ports need to be open: 7000-7002, 3840, 4005, 4808, 4810, 4995. For instructions how to open those ports, please visit portforward.com and pick your router/modem model. You can optionally DMZ your connection. It's also best to ensure no software firewall (or antivirus protection) is enabled.

Why can't I see anyone in the lobby?

This means something is wrong; you're probably disconnected shortly after. In the lobby area, there's generally a message to describe the error. Typically it'll look like "Register your nick at http://xwis.net/ga". You'll need to visit your XWIS Game Account and create a nick before continuing. If you've already created a nick, ensure you're using the "Game Password" found within your account to login to the game.

Technical Support Top

"Fatal String Error"

Right click the game icon, click "Properties" then view the "Compatibility" tab. Ensure the "Run as Administrator" option is checked. If it still persists set compatibility mode for "Windows XP (Service Pack 3)".

"TXT_PREPARECOM_FAILED"

Ensure you're using "Run as Administrator" to open the game. Optionally, you can disable UAC.

Connection (Reconnection/Disconnection) issues

Ensure the LAN patch is not in you Game Directory (file named "wsock32.dll"). Typically comes with the The First Decade or other unofficial patches.
